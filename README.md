
# Real Estate Market Dashboard

A PyQt5-based interactive dashboard for visualizing real estate market data, including price trends over time, average prices by city, and average prices by property type.

## Features
- Interactive filters for City and Property Type
- Three visualization tabs:
  - Price Trend Over Time
  - Average Price by City
  - Average Price by Property Type
- Built with PyQt5 and Matplotlib

## How to Run
1. Install dependencies:pip install pandas numpy PyQt5 matplotlib Line
2. Run the app:python dashboard.py
